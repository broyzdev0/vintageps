How to Setup Powertunnel

  1. First, download Powertunnel APP from Press here.
  2. Click the "Powertunnel" button.
  3. Copy the Powertunnel host: https://gtpshost.com/Vintageps.txt.
  4. Open the Powertunnel app, click the menu icon, and select the host settings option.
  5. Paste the copied URL. Change the host update period to "on start."
  6. Log in to your GTPS!

How to Setup Virtual Host

  1. Download and install the Virtual Host APK: Download Virtual Host APK.
  2. Download the Host file: Download Host.
  3. Open the Virtual Host app and click "SELECT HOSTS FILE."
  4. Navigate to the downloaded host file and select it.
